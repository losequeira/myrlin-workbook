# Roadmap

> Last updated: February 2026

## Shipped (v0.1.0)

- Workspace management — create, edit, organize Claude sessions
- Embedded terminal grid — up to 4 panes, drag-and-drop, resizable
- Session auto-titling from JSONL conversation logs
- Git integration — branch badges, worktree management, status indicators
- Port discovery per session with clickable links
- Cloudflare quick tunnels — one-click expose any session
- Catppuccin Mocha + Latte themes with dark/light toggle
- Click-to-position cursor in terminal panes
- Inline rename on terminal pane titles (global sync)
- Mobile-optimized layout with touch scroll and keyboard toggle
- Workspace docs panel — Notes, Goals, Tasks per workspace
- Resource monitoring — memory usage per session
- Session flags — bypass permissions, verbose mode, model selection
- SSE live updates across all views
- Password authentication with token-based sessions

## Coming Soon (v0.2.0)

- Cost tracking per session — token counting from JSONL, spend estimates
- Session status badges — needs input / working / idle / error
- Roadmap section in workspace docs panel
- Additional Catppuccin flavors — Frappe, Macchiato
- Password visibility toggle on login
- Session templates and quick launch presets
- "Start with Context" — auto-orient new sessions from project files

## Planned (v0.3.0)

- Nested workspaces — project > workspace hierarchy
- Feature tracking board — group sessions by git branch
- Auto-docs — silent background session summarizer
- Session search — full-text search across JSONL history
- Conflict detection — multi-agent file collision warnings
- Session handoff and context export

## Exploring

- QR code for tunnel / mobile pairing
- Push notifications for session events
- Subagent tracking per session
- Multi-provider support — Codex, Cursor, Aider
- Persistent tunnels at `yourname.myrlin.dev`
- Team collaboration — shared workspaces, multi-user auth
